# writing-your-own-deep-neural-network

一步步开发自己的深度学习的神经网络 Deep Neural Network。**[./dnn](./dnn/) 目录源码有详细的注释**。

[Playground](https://dnn-playground.deno.dev)

## 项目结构

```
.
├── components  # 组件
├── dnn         # 神经网络代码
├── examples    # 一些例子
├── hooks
├── islands
├── models      # 已经训练好的模型
├── routes
└── test        # 测试代码
```

## 使用

Start the project:

```
deno task start
```

This will watch the project directory and restart as necessary.
